Server & klient

Server

En server er enten hardvare eller software med en særlig rolle. En computer kan afvikle flere server-programmer af én gang, disse programmer kaldes typisk ”funktion+server”. Det kan f.eks. være:

  • Fil-server – Bruges til at opbevare filer
  • Print-server – Forbinder printere til en eller flere klienter via et netværk.
  • Web-server – Lagrer data og gør det tilgængelig for klienter. En hardware web-server kan indeholde flere software web-servere. Leverer hjemmesider via http (i en browser).
  • FTP-server – Står for file transfer protocol. Betyder at den udveksler filer til anvendelse på web-server mv. (f.eks. mellem webudviklerens computer og serveren), (f.eks. Filezilla).
  • Mail-server – Sender email fra en mailservice på nettet (f.eks. Gmail eller Hotmail).
  • Database-server – En server der modtager data fra og leverer data til web-servere og andre klienter.
  • SSH-server – Står for secure shell protocol og bruges til krypteret fjernadgang som gør at man kan styre sin computer uden at være samme sted.
  • RDP-server – Står for remote desktop protocol, denne giver også mulighed for at bruge fjernadgang på sin computer.

En server kan f.eks. være placeret i et datacenter, på et skrivebord eller i en ”dims”.

Klient

For at få adgang til en server skal man bruge en klient, det er et program der bytter information med serveren.

Adgang til serveren: For at komme ind på sin server og se/opdatere sine filer, kan man bruge en ftp-server, som f.eks. Filezilla (alternativt kan man bruge direkte filadgang), og konfigurationen kan foregå med en SSH.

Website på en server: Web-serveren leverer HTML, CSS, JavaScript og diverse media filer (f.eks. video eller billeder) til en klient (en browser), (Alle de nødvendige filer, kan findes på serveren). Serveren har en IP-adresse som kan være forbundet til et domæne. En browser kan bruge begge dele til at finde siden.
